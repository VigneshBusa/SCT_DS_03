# Bank Marketing Prediction using Decision Tree Classifier

## Project Overview
This project focuses on building a Machine Learning model using the Decision Tree Classifier algorithm to predict whether a customer will subscribe to a product or service based on demographic and behavioral data.

## Objective
To analyze customer-related data and develop a predictive model capable of identifying potential customer subscriptions.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Features
- Data preprocessing
- Label encoding for categorical variables
- Train-test data splitting
- Decision Tree Classification
- Model accuracy evaluation
- Confusion matrix generation
- Data visualization

## Dataset
Bank Marketing Dataset

## Workflow
1. Loaded and explored the dataset
2. Handled categorical data using Label Encoding
3. Split the dataset into training and testing sets
4. Trained the Decision Tree model
5. Evaluated model performance using accuracy score
6. Visualized customer subscription distribution

## Output
The model predicts customer subscription behavior and displays classification accuracy along with visual analysis.
